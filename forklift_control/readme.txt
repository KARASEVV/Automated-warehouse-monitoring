Перед вами простенькая программа, которая имитирует работу складов и погрузчиков на них.
Точка входа - файл main.py, в котором инициализируется несколько объектов Склад, на каждом из которых работает случайное
количество погрузчиков. На каждом складе есть своя очередь задач, которая случайным образом пополняется случайными
заданиями.
В текущем состоянии информация о событиях не отправляется никуда, кроме вывода в консоли. Для дальнейшего использования
вам потребуется дописать соответствующие интерфейсы вывода данных и обработать их должным образом.
Перед началом работы рекомендуется ознакомиться с кодом проекта
В файле Warehouse.py описан класс склада
в Forklift.py - класс погрузчика
в Task.py описан класс задач и очереди заданий
в config.py содержатся словари с описанием возможных маршрутов погрузчиков по складу к полкам, а также список городов
